# automated-tests
QA automation

1. Follow the installation guide at https://playwright.dev/docs/intro
2. Pull this repo to your local folder
3. Run tests with 'npx playwright test' or refer to this guide https://playwright.dev/docs/running-tests for more specific commands

Test suites are listed under E2E Automated Tests project in linear https://linear.app/gradeflow/project/e2e-automated-tests-eedf8fca8515/issues
